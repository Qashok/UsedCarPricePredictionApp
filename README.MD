# Car Price Prediction System

An end-to-end **Machine Learning project** that predicts the selling price of used cars based on key features.  
The project includes **data preprocessing, model training, evaluation, and deployment using a Flask API**.

---

##  Project Overview

The goal of this project is to build a reliable ML system that can estimate the **selling price of a car** using:
- Brand
- Fuel type
- Ownership category
- Kilometers driven

The project demonstrates a complete ML workflow from raw data to a deployable API.

---

## Models Used

- Linear Regression  
- K-Nearest Neighbors (KNN)

Models are trained, evaluated, and used for prediction via API.
